# 📚 Business Books Library (BBL)

Bienvenido a **Business Books Library**, una aplicación web moderna diseñada para explorar y descubrir los mejores libros de negocios, finanzas, liderazgo y emprendimiento.

![Business Books Library](https://placehold.co/1200x600/0d1b2a/e0e1dd?text=Business+Books+Library)

## 🚀 Características

- 🎯 **Catálogo Curado**: Libros seleccionados de las mejores categorías empresariales.
- 🔍 **Búsqueda en Tiempo Real**: Encuentra libros por título o autor instantáneamente.
- 🏷️ **Filtros por Categoría**: Explora Marketing, Finanzas, Emprendimiento, Liderazgo e Inversión.
- 📱 **PWA (Progressive Web App)**: Instalable en tu dispositivo y con soporte offline.
- 🌙 **Modo Oscuro/Claro**: Interfaz adaptable a tus preferencias visuales.
- ✨ **Diseño Premium**: UI moderna, limpia y profesional con animaciones fluidas.

## 🛠️ Tecnologías

### Frontend
- **HTML5 & Vanilla CSS**: Estructura y diseño con las últimas prácticas de CSS moderno.
- **JavaScript (Vanilla)**: Lógica rápida y ligera sin frameworks pesados.
- **Vite**: Herramienta de construcción ultra rápida.
- **Google Fonts**: Tipografías elegantes (Inter & Playfair Display).

### Backend
- **Node.js & Express**: Servidor robusto para gestionar el catálogo.
- **API Integration**: Consumo de datos dinámicos.

## 📦 Instalación y Uso

### Clonar el repositorio
```bash
git clone https://github.com/modsm54455-dot/business.git
cd business-books-library
```

### Backend
Para iniciar el servidor de datos:
```bash
cd backend
npm install
npm start
```

### Frontend
Para iniciar el entorno de desarrollo:
```bash
cd frontend
npm install
npm run dev
```

## 🌐 Despliegue

### Frontend
Puedes desplegar el frontend fácilmente en:
- **Vercel**
- **Netlify**
- **GitHub Pages**

Recuerda ejecutar `npm run build` antes de desplegar.

### Backend
Para el backend, puedes usar:
- **Render**
- **Railway**
- **Heroku**
